# Privacy Policy

Last updated: 2026-05-04

`Reversi Friend` is a local-only Android reversi game.

## Data Collection

This app does not collect, store, send, or share personal information.

## Network Access

This app does not use network communication.

## Ads and Payments

This app does not include ads, in-app purchases, subscriptions, or login features.

## Game Data

Current game state is handled only inside the app while playing. It is not sent to any server.

## Images

Character images are included as local app assets. Before public store release, confirm that all images are original or legally usable for distribution.

# スクリーンショット作成メモ

GitHubとitch.ioに載せるスクリーンショットは、最初は3枚で十分です。

## 撮る画面

1. メニュー画面
   - CPUの強さ、自分の色、キャラ選択が見える画面
2. 対局中の画面
   - 盤面、石、会話エリアが見える画面
3. 結果画面
   - 勝利、敗北、引き分けの中央表示が見える画面

## スマホで撮る方法

多くのAndroid端末では、以下で撮れます。

```text
電源ボタン + 音量下ボタン
```

撮った画像は、スマホの `写真` や `ギャラリー` に入ります。

## PCに移す方法

簡単な方法:

- 自分宛てにLINEやメールで送る
- Google Driveにアップロードする
- USBケーブルでPCにコピーする

## ファイル名の例

```text
screenshot_menu.png
screenshot_game.png
screenshot_result.png
```

あとで `README.md` や itch.ioページに貼ります。

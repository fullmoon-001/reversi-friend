## Description

```text
Reversi Friendは、Android向けの1人用リバーシゲームです。

リカとマイのキャラクター会話を見ながら、CPUと対局できます。
CPUの強さ、自分の色、自分のキャラを選んで遊べます。

特徴:
- 8x8の標準的なリバーシ
- 黒・白の選択
- CPUの強さを3段階から選択
- リカ / マイのキャラクター選択
- 戦況に応じたキャラクター会話
- 勝利・敗北・引き分け表示
- 広告なし
- 課金なし
- ログインなし
- ネット通信なし

インストール方法:
1. APKをAndroidスマホにダウンロードします。
2. ファイルを開きます。
3. 端末の案内に従ってインストールします。
4. 「提供元不明のアプリ」の許可を求められた場合は、許可してから再度インストールします。

注意:
このアプリはGoogle Play版ではありません。
個人制作のポートフォリオ・テスト公開版です。
```

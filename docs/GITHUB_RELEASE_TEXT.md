# Reversi Friend v0.1.0

Android向けの1人用リバーシゲームです。

## 内容

- 8x8の標準的なリバーシ
- 黒・白の選択
- CPUの強さを3段階から選択
- リカ / マイのキャラクター選択
- 戦況に応じたキャラクター会話
- 勝利・敗北・引き分け表示
- 広告なし
- 課金なし
- ログインなし
- ネット通信なし

## ダウンロード

`reversi-friend-v0.1.0.apk` をAndroidスマホに入れて起動してください。

## 注意

Google Playからのインストールではないため、端末によっては「提供元不明のアプリ」を許可する必要があります。  
このAPKはローカル配布確認用です。Google Play公開用の正式署名版ではありません。
